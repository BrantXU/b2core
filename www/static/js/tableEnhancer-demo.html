使用方法

只需一行代码即可将传统表格转换为增强表格：

const enhancer = new TableEnhancer('demoTable', {
    pageSize: 10,
    searchable: true,
    sortable: true
});

配置选项

  • pageSize: 每页显示条数，默认10
  • pageSizes: 可选的每页显示条数数组，默认[10, 20, 50, 100]
  • showInfo: 是否显示记录信息，默认true
  • showPagination: 是否显示分页，默认true
  • searchable: 是否启用搜索，默认true
  • sortable: 是否启用排序，默认true
  • searchPlaceholder: 搜索框占位符，默认'搜索...'
